Cyber Cafe Management System

Overview: The Cyber Cafe Management System is a web application designed for managing a cyber cafe's operations,
 including user registration, cabin allocation, bill generation, and tracking of frequent users.

Features:
--User Registration
--View All Users
--Allocate Cabins
--Generate Bills
--Identify Priority Users

Technologies Used:
--Python
--Flask
--SQLite
--HTML/CSS

Prerequisites:
--Python 3.x
--Flask
--SQLite
